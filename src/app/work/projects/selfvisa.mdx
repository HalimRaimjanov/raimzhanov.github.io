## Overview

In this project, I developed a website for SelfVisa, a leading visa center specializing in U.S., European, and Schengen visas. The goal was to create a streamlined, user-friendly platform that offers services like visa consultation, document preparation, and interview coaching.

## Key Features
- **Multi-page Design: Built with several pages, including service offerings, pricing, and client support.
- **Client Interaction: Features to schedule consultations and FAQs for a better customer experience.
- **Modern Design: Applied modern web design principles for a responsive, engaging experience across all devices.

## Technologies Used
- **HTML & CSS: For a responsive and visually appealing layout.
- **Query: To enhance interactivity and manage dynamic content.

## Outcome

The website successfully reflects the professionalism of SelfVisa, making it easier for users to navigate and apply for their desired visas with confidence.
